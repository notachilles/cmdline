
The error message was redirected to error.txt using file descriptor 2.

Linux manages file access using file descriptors. Every opened file,
device, or stream is assigned a file descriptor number. Processes use
these descriptors to perform read and write operations.

Standard streams are:
0 - stdin
1 - stdout
2 - stderr

Redirection changes where these streams send or receive data.
Resource limits prevent processes from consuming unlimited system
resources.
